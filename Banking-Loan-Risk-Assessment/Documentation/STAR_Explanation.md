# STAR Interview Explanation

## Situation
Retail lenders need faster loan decisions, better default monitoring, and branch-level visibility, but raw transaction data is often scattered across operational systems.

## Task
I set out to build a full analytics solution that could store banking data, analyze approval and default patterns, generate predictive insights, and present them through business dashboards.

## Action
I designed a normalized SQL database, generated realistic banking data, wrote analytical SQL queries, and developed Python pipelines for cleaning, EDA, feature engineering, and machine learning. I then mapped the transformed data into a Power BI star schema and defined DAX measures for executive and operational reporting.

## Result
The final project demonstrates an end-to-end lending analytics workflow with production-style design choices. It is portfolio-ready for a fresher, while still showing database modeling, analytics engineering, machine learning, and dashboard development depth.
