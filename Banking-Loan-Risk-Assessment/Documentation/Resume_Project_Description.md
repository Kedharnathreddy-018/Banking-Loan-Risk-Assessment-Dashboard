# Resume Project Description

**Banking Loan Risk Assessment & Analytics Dashboard**

- Built an end-to-end banking analytics project using SQL, Python, and Power BI to analyze 10,000 customer profiles, 15,000 loan applications, and 50,000 repayment transactions.
- Designed a normalized relational database, wrote 25+ business SQL queries with CTEs, window functions, views, procedures, and triggers for credit risk and branch performance analysis.
- Engineered borrower risk features such as debt-to-income ratio, payment compliance score, and customer risk score, then trained Logistic Regression, Random Forest, and XGBoost models for approval and default prediction.
- Designed a multi-page Power BI dashboard covering executive KPIs, portfolio trends, high-risk customers, and branch-level revenue insights.
- Documented the architecture, data model, ETL flow, business recommendations, and interview-ready STAR explanation to present the project as a production-style analytics portfolio case study.
